How to run
=====
To get this project working, perform the minimum steps in Maven:

>$ mvn groovy:compile

>$ mvn groovy:execute

You may also need the following: 

>$ export MAVEN_OPTS="-Xmx1024m -XX:MaxPermSize=256m -XX:PermSize=128m"
